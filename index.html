About

A builder who starts at the hardware.

Hossam is a Computer Science and Artificial Intelligence student at New Mansoura University with a habit of understanding systems from the bottom up — digital logic, assembly instruction sets, and object-oriented software design — before applying that grounding to machine learning.

Problem & goal

Clinicians and patients often lack a fast, data-backed first read on symptoms. The goal was a decision-support tool that could suggest likely conditions from reported symptoms, accurately enough to be genuinely useful as a first pass.

Solution

A neural network trained on a 41-disease, 117-symptom Kaggle dataset, wrapped in knowledge-based reasoning and optimization logic, then exposed through a REST API and an interactive analytics dashboard.

Role & approach

  • End-to-end build: data preprocessing, feature engineering, model tuning
  • Designed the REST API and full-stack architecture
  • Applied formal software engineering methodology for scalability

Result

A working full-stack platform achieving 99.46% test accuracy, demonstrating that a student-built system can combine ML rigor with genuine software engineering discipline.
